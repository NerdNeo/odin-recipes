Odin Recipes

Greetings! This page is my interpreation of the first real project in The Odin Project's "Foundations" curriculum.

This project is to create a simple recipe book with HTML. The only specifications are that it is to "consist of a main index page which will have links to a few recipes."

This page itself looks pretty crappy because it's made with only HTML, however, the module right after this project taught the basics of CSS in a really awesome way, one of the projects in that module was to apply some CSS to the previous project, and I thought it would be cool to link to that version HERE, I feel like it shows just how different a page can look with some very basic CSS skills. With that said, if you're looking at this on mobile it will all look bad because no responsive design has been built into this webpage. I digress...
